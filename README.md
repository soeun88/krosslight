# krosslight

### 프로젝트:
- 무신호 횡단보도 내 사고율 감소를 위한 프로젝트

### 목표:
-  보행자와 차량을 인식하여 충돌 위험을 감지하고 알리는 시스템
  
### 주요 업무:
- 보행자/차량 이미지 수집
- roboflow를 활용하여 이미지 라벨링 및 어노테이션
- yolov11n 모델을 활용하여 이미지 학습
- cv2를 황용해 학습한 모델 테스트


### 사용 기술:
- yolov11n
- python
- roboflow
- openCV
